## 🧠 NEXUS-7: Dark Choices

An immersive Horror Sci-Fi Interactive Story Game built using HTML, CSS, and JavaScript, where player decisions shape the narrative, influence stats, and unlock multiple endings in a branching storyline set inside a mysterious research facility.

## 🚀 Features
* **Interactive Story Engine** – Dynamic narrative driven by player choices.
* **Multiple Endings System** – 10+ unique endings based on decisions and stats.
* **Stat-Based Gameplay** – Health, Intelligence, and Trust affect outcomes.
* **Branching Decision Paths** – Choices lead to different story routes.
* **Inventory System** – Collect and use items to unlock paths.
* **Typewriter Text Animation** – Cinematic storytelling with progressive text reveal.
* **Keyboard Controls Support** – Use keys (1–4, space) for faster gameplay.
* **Save & Resume System** – Continue previous runs using stored data.
* **Audio Engine** – Background ambience and sound effects for immersion.
* **Dynamic Scene Transitions** – Smooth fade effects between scenes.
* **Stat Progress Bars** – Real-time visual updates for player stats.
* **NPC Dialogue System** – Character interactions with styled dialogue boxes.
* **Glitch & Horror UI Effects** – Sci-fi aesthetic with glitch animations.

2. **Open the project and navigate to the file**:
```

```
3. **Run the game in any modern browser**

## 🎮 How to Use
**Start the Game**
* Enter your name
* Click INITIALIZE

**Resume Previous Game**
* If save exists → click RESUME PREVIOUS RUN

**Play the Story**
* Read story text
* Click choices to progress

**Manage Inventory**
* Items collected appear in inventory
* Some choices require specific items

**Save Game**
* Click Save icon in top bar

**Restart Game**
* Click Restart icon
